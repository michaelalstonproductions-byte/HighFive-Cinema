# HighFive Cinema Privacy Notice Draft

This draft is a launch operations artifact and requires legal review before public release.

## Data Categories

HighFive Cinema may process:

- Account profile data
- Creator profile data
- Content publishing metadata
- Viewer library state
- Playback progress
- Search and discovery events
- Analytics events
- Notification preferences
- Purchase and entitlement state
- Moderation and support records

## Data Use

Data is used to provide streaming, creator publishing, discovery, library sync, analytics, support, safety, rights enforcement, and entitlement checks.

## Analytics

Analytics events should avoid unnecessary personal data and should use authenticated or anonymous identifiers appropriate to the user state.

## Creator Content

Creator-submitted content and metadata may be reviewed for publishing, distribution, rights, safety, and platform operations.

## Notifications

Users may receive product notifications for publishing updates, new episodes, creator activity, system events, and account-related events when enabled.

## Account Deletion and Export

Users may request account data export and account deletion through supported product workflows.

## Third Parties

Production launch may require Apple services, backend hosting, object storage, media processing, monitoring, and notification delivery providers.

## Manual Review Required

Before public release, the final privacy notice must match the configured App Store privacy answers, PrivacyInfo manifest, backend behavior, analytics payloads, and support operations.
