# Creator Studio Control Center Backend Plan

## Goal

Make Creator Studio a control center for all local and future backend-powered tools.

## Modules

- Current Project
- Asset Board
- Caption Lab
- Social Media Kit
- Instagram Connect readiness
- VOD Package
- Release Checklist
- Provider Readiness

## Backend records

- creator_projects
- creator_assets
- social_kits
- social_posts
- social_platform_status
- vod_packages
- vod_checklist_items
- communication_records
- analytics events from allowlist only

## Safe actions now

- Review Social Kit
- Preview VOD Package
- Save Local Draft
- Back to Profile

## Live actions later

- Connect Instagram
- Submit package
- Publish post
- Send to distributor

These actions require separate provider, auth, backend, privacy, and support approval.
